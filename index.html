🎓 青年大学习：MVP 构建清单与项目构想

为年轻人补上生活中缺失的一课，打造一站式生活技能学习平台

🔍 一、项目简介

🧠 初衷

大学生和刚步入社会的年轻人普遍缺乏生活技能和社会事务处理经验，本项目旨在打造一个面向年轻人的「生活技能百科平台」，通过关键词搜索 + 内容总结，快速获取衣食住行、社保医保、租房体检等生活常识的图文指南。

🚀 二、MVP 功能清单（最小可行产品）

✅ 1. 关键词搜索入口

功能：用户输入如"怎么挑西瓜""北京体检流程"关键词。

技术实现：

  • SpringBoot 接口接收搜索词
  • 关键词预处理（去停用词、拼写修正）

✅ 2. 网络内容爬取模块

功能：从主流平台获取相关内容（知乎、小红书、B站等）。

技术实现：

  • 使用 Python 脚本调用爬虫框架（如 requests + BeautifulSoup 或 Playwright）
  • 限制爬取 2~3 个网页/视频链接，模拟内容检索

✅ 3. 内容总结生成

功能：将爬取到的内容交给 AI 总结为图文知识卡片。

技术实现：

  • 调用 ChatGPT API（或文心一言/通义千问等）传入网页原文 → 返回结构化总结
  • 输出：关键词 + 总结段落 + 原始链接 + 标签分类（如「挑选技巧」「地区差异」）

✅ 4. 前端展示页面（简单 HTML 或 Vue 模板）

功能：展示搜索结果 + 图文总结 + 外链原文

页面布局建议：

  • 搜索框（顶部）
  • 卡片区：每张卡片显示关键词、内容概要、来源平台、原文链接
  • 标签筛选（可选）

✅ 5. 地区化数据演示（选做）

功能：演示地区差异，比如蔬菜价格、租金水平

实现方式：

  • 写死一组示例数据（如 北京 vs 贵阳）
  • 使用 chart.js 或 echarts 展示可视化折线图/柱状图

🧱 三、项目结构构想（技术栈）

后端服务

部署
